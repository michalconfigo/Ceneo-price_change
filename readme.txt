== Description ==

Ceneo.pl – oficjalna wtyczka dla WooCommerce. Pełna integracja sklepu z Ceneo: plik XML z produktami, Zaufane Opinie, Kup Teraz. 

Ceneo to największa porównywarka cen w Europie, dzięki której możesz szybko pozyskać nowych klientów i znacząco zwiększyć swoją sprzedaż. Współpracuje z nami 26 tys. sklepów, a zakupy w nich robi aż 23 mln użytkowników miesięcznie. Skutecznie wspieramy sklepy internetowe od 16 lat, oferując średnią konwersję na poziomie 5,5%. To dlatego już co 8. transakcja online w Polsce odbywa się właśnie na Ceneo.

== Frequently Asked Questions ==

= Czy jest to wtyczka wspierana przez Ceneo? =

Tak, wtyczka jest całkowicie kompatybilna z Ceneo, ponieważ jest naszego autorstwa. Wtyczka będzie rozwijana i dostosowywana do ewentualnych zmian w funkcjonowaniu platformy Ceneo

= Czy wtyczka jest bezpłatna? =

Tak, możesz z niej korzystać bez żadnych opłat. 

= Mam pytanie, na które nie mogę znaleźć odpowiedzi. Z kim mogę się skontaktować? =

Wyślij wiadomość na adres wsparciesklepu@ceneo.pl


== Screenshots ==

1. Znajdź i zainstaluj wtyczkę
2. Połącz kategorie w swoim sklepie z kategoriami Ceneo
3. Skonfiguruj właściwości generowania XML z ofertami, wprowadź GUID by zbierać opinie i wprowadź klucz API Ceneo by automatycznie pobierać transakcje w Kup Teraz
4. Obsługuj transakcje Kup Teraz bezpośrednio w panelu WooCommerce. Zostaną pobrane automatycznie kiedy wprowadzisz swój klucz API
